Neste trimestre as tags das quais aprendemos foram as:

Tag para imagem:

Descrição

Para colocarmos uma imagem no nosso html, vocẽ precisa usar a tag "img src", lembre-se de por < e >, um no incio e um no final, após por a tag, coloque o simbolo de igual (=) e após isso, coloque as aspas, dentro das aspas você deve colocar o endereço da imagem, logo a tag ficara como na foto acima.

Tag para ítem:

Descrição

Esta tag serve para você aplicar ítens ao seu HTML, para aplicar a tag você começara usando as setas das quais foram ditas na descrição anterior, após isso use as siglas UL e após isso, colocara a tag LI, dentro desta tag nós iremos por os itens que queremos, Ex:

  • ítem 1
  • ítem 2

E por fim, fecharemos a tag com <ul> . Também podemos trocar a tag UL para a OL.

Tag para divisórias:

Descrição

Nós usaremos a tag <div> para separarmos conteúdos no nosso HTML, podemos usar elas também para separarmos em classes, um exemplo seria "<div class="">", no CSS conseguiremos mudar as cores e estilo sem mexer no restante do site.

Tag de Ancoragem:

Descrição

A tag <a href=""> nome do seu arquivo </a> vai te guiar para uma outra pagina, porém precisaremos de um botão para isto. Para alterarmos o estilo do nosso href, devemos por "a" no style (poucos sabem dessa informação).
