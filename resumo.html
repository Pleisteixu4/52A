Tag para imagem:

Descrição

Para colocarmos uma imagem no nosso html, vocẽ precisa usar a tag "img src", lembre-se de por < e >, um no incio e um no final, após por a tag, coloque o simbolo de igual (=) e após isso, coloque as aspas, dentro das aspas vocé deve colocar o endereço da imagem, logo a tag ficara como na foto acima.

Tag para ítem:

Descrição

Esta tag serve para você aplicar ítens ao seu HTML, para aplicar a tag você começara usando as setas das quais foram ditas na descrição anterior, após isso use as siglas UL e após isso, colocara a tag LI, dentro desta tag nós iremos por os itens que queremos, Ex:

  • ítem 1
  • ítem 2

E por fim, fecharemos a tag com < / ul >
